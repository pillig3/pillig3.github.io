Mandelbrot Set Explorer

This is a simple program, written for my Software Design class, that allows the user to explore the Mandelbrot set. Zoom in by clicking, and move around using the arrow keys or WASD. The user may also manually choose the location they wish to see by entering a complex number and screen radius. These images are generated using the escape time algorithm, and the user may choose how many iterations the map will be applied to each point before deciding whether or not it escapes. A low number of iterations will result in faster rendering times, but a fuzzier and less accurate picture. It's probably best to use a small number (~\(1000\)) while moving to the desired location, and a larger number to then render the image.

Scheme Interpreter

This is an interpreter for the R5RS standard of Scheme, written in C. I made this with another student for a Programming Languages class; it's not a "proper" Scheme interpreter, as it's missing some required features, but it's pretty good. It has an REPL (press CTRL-D to exit) and can handle all forms of variadic lambda expressions (as described here). The main missing features are tail recursion and better garbage collection — these are works in progress. :)
