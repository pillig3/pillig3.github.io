Mandelbrot Set Explorer

This is a simple program, written for my Software Design class, that allows the user to explore the Mandelbrot set. Zoom in by clicking, and move around using the arrow keys or WASD. The user may also manually choose the location they wish to see by entering a complex number and/or screen radius. These images are generated using the escape time algorithm, and the user may choose how many iterations the map will be applied to each point before the program decides whether or not it escapes. A low number of iterations will result in faster rendering times, but a fuzzier and less accurate picture. It's probably best to use a small number (~\(1000\)) while moving to the desired location, and a larger number to then render the image.
