Pascal's Triangle



Pascal's Triangle Mod 17

Pascal's triangle, where each number is a hexagon colored by its residue mod 17. 0 mod 17 is black, and larger numbers are brighter red. Note the structure: each small triangle has a side length of 17, after which comes an upside-down triangle where everything is divisible by 17. This can be thought of as a mod-17 version of the Sierpinski gasket, which is obtained by coloring Pascal's triange mod 2. This was made in Mathematica, with the following code (the number 17 can be changed to any other number to get more neat results - composite numbers are especially interesting).

n = 17;
hex = ({{Cos[Pi/6], -Sin[Pi/6]}, {Sin[Pi/6], Cos[Pi/6]}}.#) & /@ CirclePoints[6];
Show[Table[Graphics[{RGBColor[Mod[Binomial[m, k], n]/(n - 1), 0, 0], Polygon[({(-Sqrt[3] m)/2 + Sqrt[3] k, (-3 m)/2} + #) & /@ hex]}], {m, 0, n^2}, {k, 0, m}]]
